**SOFTWARE DE ORGANIZAÇÃO - TECHFLOW SOLUTIONS**
Inicialmente uma ideia buscando a melhoria da empresa TechFlow Solutions, com objetivo de desenvolver um sistema que permite acompanhar o fluxo de trabalho em tempo real.


**Informações**
Criação de um sistema web, realizado com a utilização das linguagens Python e HTML, com o intuito de fornecer uma pagina minimalista e objetiva para a empresa "Techflow Solutions". Seu foco é um sistema de organização que utiliza fundamentos dos metodos agéis, tendo destaque em seu quadro Kanban. Assim, possiblitando mais clareza para os objetivos que devem ser atingidos pela empresa.


**Foco**
Realizar um software para o gerenciamento de tarefas com enfase nos conceitos da diciplina de Engenharia de Software, onde seus objetivos são:

- Dominio do sistema Git/GitHub com o uso de commits;
- Integração Contínua CI/CD;
- Criação de um CRUD
- Metodos Ágeis com enfase no Kanban;
- Execução de workflows.
- Metodo de verificação da entegridade do codigo
- Organização nas pastas do
- Mudanças no Escopo do projeto;


**Metodologia Ágil**
Quadro Kanban:

- Realização de mudanças atualizadas no projeto
- Melhor vizualição do que esta sendo executado
- Controle no teste e qualidade do software
- Quadro visual no GitHub Projects.
- Fundamental para organização antes da inicialização de um projeto

**PROJETO:**
Utilizar a versão Python 3.9 ou mais recente junto ao Git

```bash
git clone https://github.com/samuelgit2/projeto_software
cd projeto_software
pip install -r requirements.txt
python src/app.py
pytest tests/ -v
```

Abrir o navegador e acessar: http://localhost:5000
